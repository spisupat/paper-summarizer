Web Content Summarizer

This tool summarizes web content using OpenAI's o4-mini model and copies the summary to your clipboard as markdown.

Bookmarklet

Drag this button to your bookmarks bar:

Summarize

How to use:

  1. Drag the "Summarize" button to your bookmarks bar
  2. Navigate to any webpage you want to summarize
  3. Click the "Summarize" bookmark
  4. The summary will be copied to your clipboard in markdown format
